## How to Run Java Projects

### Clone Repository

```bash
git clone https://github.com/Kamal1506/cts_digital_nurture_5.0.git
```

### Navigate to Project

```bash
cd cts_digital_nurture_5.0
```

### Compile

```bash
mvn clean compile
```

### Run Tests

```bash
mvn test
```
